Au cours de cette formation nous allons avoir 2 matières

L'intégration

Le HTML:

Le HTML sert à créer le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, ect.

Le développement

Le Javascript

Javascript est un langage de programmation de script employé pour rendre une page web intéractive, le comportement de la page et son apparence pourra varier en fonction de certain paramètres

le PHP

Le PHP est un langage de programmation libre, utilisé pour généreé des pages web dynamiques, qui sont donc modifiés en fonction de certaines informations

MySQL

MySQL est un systéme de gestion de bases de données, il permet d'enregistrer des informations rempli par l'utiolisateur (exemple : formulaire de création de compte pour ensuite être ré-utiulisé au moment
